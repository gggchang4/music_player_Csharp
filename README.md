# MusicPlayerApp

[简体中文](README.zh-CN.md) | [English](README.en.md)

A local desktop music player client built with C#, WPF, and .NET Framework 4.8.

## Repository

- GitHub: [gggchang4/music_player_Csharp](https://github.com/gggchang4/music_player_Csharp.git)
- Author: GGG Chang

## Overview

MusicPlayerApp is a Windows music player focused on local audio playback and library management. The project includes music import, playlist management, favorites, search, lyric display, user settings, and a SQLite-based local data layer.

## Highlights

- Local audio import and playback
- Song / artist / album browsing
- Playlist and favorites management
- Search across songs, artists, and albums
- Local `.lrc` lyric display
- User profile, theme, and playback settings
